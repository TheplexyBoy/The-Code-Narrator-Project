# Digit Counter Python Script

## Description
This project is a simple Python program that counts the number of digits in a number entered by the user.  
It is an educational example designed to demonstrate basic programming concepts such as variables, loops, and arithmetic operations.

## How it works

1. The program asks the user to enter a number.
2. The number is stored in a variable called `number`.
3. The program uses a `while` loop to check if the number is greater than zero.
4. The algorithm divides the number by ten in every iteration.
5. The counter variable increases every time the loop runs.
6. The program prints the total number of digits at the end.

The program uses simple logic to process the number and count its digits.  
It demonstrates how loops and variables work in Python.  
The system processes the number step by step until it becomes zero.
